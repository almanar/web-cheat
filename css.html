Selectors

There are 3 ways to use CSS:

  • #external
  • #internal
  • #inline

#pseudo element

Pseudo elements are selectors for things that are not real HTML elements, such as the first letter of the content of a paragraph.

Pseudo elements are specified as tag:pseudo-element.

Pseudo element styles cannot be specified inline.

first letter

#pseudo classes

Classes that are not really classes such as a:visited, a:link for links.

#asterisk #*

*: all elements recursively

#scoped

Very low browser support as of 2013 but great thing: sets scope of a style block in the body to only the parent element.
